# PWPush-Password-Generator
Powershell Script to Push Passwords Using Powershell
![alt text](images/image.png)

This is a modified version of the original PWPush Password Generator from Tachaeon at https://github.com/Tachaeon/PWPush-Generator

The original lacked some features I wanted to see in the app and modified the original code to inlude the following.

1 - Added Custom Password Generation.
2 - Added Random Password Options.
3 - Change the Generate / Push button and Split it out to seperate buttons.
